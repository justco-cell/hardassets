HardAssets.io — Free Portfolio Dashboard for Gold, Silver, Real Estate, Crypto & Hard Asset Investors

What is HardAssets.io?

HardAssets.io is a free portfolio tracking dashboard built for investors who hold gold, silver, real estate syndications, cryptocurrency, private notes, and collectibles. Track everything that holds value — with live spot prices, risk scoring, allocation analysis, and secure cloud sync.

Unlike Hard Assets Alliance (a precious metals dealer), HardAssets.io does not sell any financial products. It is a pure tracking and analytics tool — completely free, no credit card required.

Track These Asset Classes

  • Precious Metals — Gold, silver, platinum, palladium with live spot prices and oz-per-unit conversion
  • Real Estate Syndications — LP positions with rate tracking, projected IRR, and sponsor management across 16 deal types
  • Physical Real Estate — Properties with equity, cash flow, cap rate, and mortgage tracking
  • Cryptocurrency — 20+ coins with live CoinGecko prices, cost basis, and gain/loss tracking
  • Notes & Lending — Hard money loans, private notes, P2P lending with interest income tracking
  • Collectibles & Art — Watches, wine, art, cars, jewelry with valuation and gain tracking

Key Features

  • Live metal prices from metals.dev API
  • Live crypto prices from CoinGecko for 20+ coins
  • Live currency exchange rates for 16 currencies
  • Portfolio allocation with target comparison
  • Risk scoring (1-10) with color coding
  • CSV import and export on every tab
  • Secure cloud sync via Google Sign-In
  • Deal analyzer with cap rate, CoC, and DSCR calculations
  • Activity log tracking all portfolio changes
  • Works on desktop and mobile — installable as a PWA

Frequently Asked Questions

Is HardAssets.io free?

Yes, completely free. No credit card, no premium tier. All features are included.

Is HardAssets.io the same as Hard Assets Alliance?

No. HardAssets.io is a separate portfolio tracking tool. Hard Assets Alliance sells precious metals. HardAssets.io does not sell anything — it helps you track investments you already own.

How is my data secured?

Data is stored in encrypted PostgreSQL on Supabase. Your browser never touches the database directly. Authentication uses Google Identity Services.
